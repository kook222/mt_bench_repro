# MT-Bench 카테고리 구조

MT-Bench는 **80문항을 8개 카테고리로 구성** (각 10문항). 모든 문항은 2-turn 구조 (follow-up 질문 포함).

---

## 카테고리 설명

| 카테고리 | 유형 | 예시 문항 |
|---------|------|---------|
| **writing** | 창작, 에세이, 이메일 작성 | "하와이 여행 블로그 포스트를 작성하라" |
| **roleplay** | 특정 인물/역할 가정 대화 | "엘론 머스크처럼 대화하라" |
| **extraction** | 정보 추출, 요약, 분류 | "영화 리뷰를 1~5점으로 평가하고 JSON으로 출력하라" |
| **reasoning** | 논리 퍼즐, 추론 | "레이스에서 2등을 추월하면 몇 등인가?" |
| **math** | 수학 계산, 증명 | "삼각형 넓이와 외접원 반지름을 구하라" |
| **coding** | 프로그래밍 문제 | "디렉토리 내 텍스트 파일에서 top-5 단어를 추출하라" |
| **stem** | 과학/공학 개념 설명 | "양자역학의 중첩 원리와 얽힘을 설명하라" |
| **humanities** | 역사, 경제, 철학 | "GDP, 인플레이션, 실업률의 상관관계를 설명하라" |

---

## Hard vs Easy 분류 (논문 기준)

논문은 카테고리를 난이도에 따라 두 그룹으로 나눈다:

| 그룹 | 카테고리 | 근거 |
|------|---------|------|
| **Hard** | math, reasoning, coding | 정답이 명확하고 모델 간 점수 차이가 큼 |
| **Easy** | writing, roleplay, humanities | 주관적이고 대부분의 모델이 높은 점수를 받음 |
| (중간) | extraction, stem | 중간 난이도 |

## Reference-guided 채점 대상

math / reasoning / coding 3개 카테고리만 정답(reference)이 제공되며, 이를 judge 프롬프트에 포함해 채점 정확도를 높인다.

> 논문 Table 4: 정답 없이 채점 시 GPT-4도 70% 오류 → 정답 포함 시 15%로 감소.
